If you scan long source files with a phone camera, you usually **scroll manually** between each capture. On office laptops you often **cannot install** browser extensions or desktop tools.

**CamToCode Scroll Automation** solves this with a free page at **[camtocode.com/scroll](https://camtocode.com/scroll)**.

## What it does

- Opens a **local file** from your computer (.py, .js, .ts, etc.)
- Scrolls in **fixed line sections** (auto-detect screen lines or set manually)
- **Keyboard mode:** S/N next · P pause · T stop · F fullscreen
- **Timer mode** syncs with CamToCode **Auto Re-capture** on your phone
- **No login, no API** — file never leaves your browser

## Who benefits

- Developers on **corporate laptops** (no software install)
- Students scanning **long assignment files**
- Interview prep from **restricted machines**
- Anyone capturing multi-page code with [CamToCode](https://camtocode.com)

## How to use with CamToCode

1. Laptop: [camtocode.com/scroll](https://camtocode.com/scroll) → open file → fullscreen
2. Phone: [camtocode.com/app](https://camtocode.com/app) → crop ROI → enable Auto Re-capture
3. Capture each section; scroll advances on laptop automatically or via S/N keys

Full guide: [camtocode.com/docs](https://camtocode.com/docs)

## Offline use

Load the page once online, then you can turn off Wi‑Fi and still scroll — as long as you don't refresh the tab.
